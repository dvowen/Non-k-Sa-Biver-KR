# 아카라이브 게시본 이미지·버전 주소 보강

## 목적

현재의 단순한 커뮤니티 게시글 형태는 유지하면서, 글 상단에서 게임 화면과 일반판·치트판 주소를 한 번에 확인할 수 있게 한다.

## 확정 구성

1. 제목과 게임 소개 문단은 그대로 둔다.
2. 소개 문단 바로 아래에 기존 아카라이브 스크린샷 4장을 원래 순서대로 세로 배치한다.
3. 네 번째 이미지 다음에 `오리지널`, `치트` Base64 주소를 차례로 둔다.
4. 두 주소는 링크로 만들지 않으며, 모바일에서도 카드 밖으로 넘치지 않게 줄바꿈한다.
5. 기존 조작법, 세이브 백업, CG 공략 내용과 단순한 흰색 게시글 형식은 유지한다.
6. 일반판과 치트판의 게시용 HTML에 같은 이미지 순서와 두 버전 주소를 반영한다. 치트판 전용 사용법과 각 빌드의 세이브 관리 주소는 그대로 유지한다.

## 검증

- 이미지 4개의 순서와 `src` 존재 여부를 정책 테스트로 고정한다.
- 일반판·치트판 Base64 주소가 두 게시본에 모두 들어 있는지 확인한다.
- 직접 이동 가능한 외부 `<a>` 링크는 허용하지 않는다.
- 아카라이브 엄격 린트에서 오류와 경고가 없어야 한다.
- 데스크톱과 모바일 브라우저 렌더링에서 이미지 폭과 Base64 줄바꿈을 확인한다.
