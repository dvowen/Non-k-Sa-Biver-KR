# Non-k Sa-BIVER v0.7 User Docs Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** v0.6에서 v0.7로 달라진 내용을 스포일러까지 포함해 설명하는 패치노트와 통합 공략집을 일반판·치트판에 각각 PR로 배포한다.

**Architecture:** 일반판 브랜치에서 사실 계약 테스트와 Markdown/아카라이브 HTML 문서를 먼저 완성한다. 초안은 `humanize-korean` Fast 절차로 구간별 윤문하고, 보호 대상 검증을 통과한 최종본만 루트 문서로 둔다. 치트판은 일반판 문서 커밋을 이식한 뒤 배포 URL과 관리자 도구 안내만 치트판에 맞게 조정한다.

**Tech Stack:** Markdown, HTML `<details>`/인라인 CSS, Node.js `node:test`, Git/GitHub CLI
