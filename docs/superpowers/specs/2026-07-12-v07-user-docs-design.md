# Non-k Sa-BIVER v0.7 유저 문서 설계

## 목표

v0.6에서 v0.7로 넘어오며 달라진 내용을 플레이어가 바로 이해할 수 있는 상세 패치노트로 정리한다. 기존에 아카라이브 게시용으로 작성했던 조작법·CG 공략도 복원해 v0.7 기준으로 갱신한다. 사용자가 스포일러 전체 공개를 승인했으므로 성인 이벤트의 해금 조건과 공/수 분기도 숨기지 않는다.

## 근거 자료

- `v0.6-kr-cheat.1`과 현재 v0.7 코드·번역 데이터의 차이
- 기존 기술 문서 `PATCH_NOTES_v0.7.md`
- 2026-05-24 세션에 남은 v0.6 아카라이브용 조작법 및 캐릭터별 CG 공략
- v0.7 게임 코드의 이벤트 조건, 갤러리 플래그, 아이템 정의

확인되지 않은 개발자 의도나 밸런스 평가는 사실처럼 적지 않는다. 코드에서 직접 확인한 조건과 플레이에 필요한 설명을 분리해 쓴다.

## 문체와 윤문

초안은 사실을 빠짐없이 담는 데 집중한다. 최종본은 `humanize-korean` Fast 절차로 윤문한다. 고유명사, 수치, 버전, 이벤트명, 선택지, 가격은 보호 대상으로 두며 변경하지 않는다. 5,000자를 넘는 공략집은 조작법·캐릭터 공략·신규 장비 섹션으로 나눠 윤문한 뒤 합친다.

윤문 후에는 번역투, 기계적인 병렬 구조, 같은 종결 반복, 접속사 남발을 확인한다. 사실 보존, 변경률 30% 이하, 장르와 격식 유지, S1 패턴 제거, 임의 수사 미추가까지 자체검증한다.

## 검증

- 패치노트와 공략집의 고유명사·수치·이벤트 조건을 코드와 다시 대조한다.
- Markdown 링크와 헤딩 구조를 검사한다.
- 아카라이브 HTML의 `<details>`·`<summary>` 짝과 인라인 스타일을 검사한다.
- `git diff --check`로 공백 오류를 확인한다.
- 초안과 최종본의 보호 대상이 달라지지 않았는지 비교한다.
